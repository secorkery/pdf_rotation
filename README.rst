============
PDF Rotation
============


.. image:: https://img.shields.io/pypi/v/pdf_rotation.svg
        :target: https://pypi.python.org/pypi/pdf_rotation

.. image:: https://img.shields.io/travis/secorkery/pdf_rotation.svg
        :target: https://travis-ci.com/secorkery/pdf_rotation

.. image:: https://readthedocs.org/projects/pdf-rotation/badge/?version=latest
        :target: https://pdf-rotation.readthedocs.io/en/latest/?version=latest
        :alt: Documentation Status




This package determines the orientation of a PDF document (north, south, east, west) and return the number of degrees to rotate the document so it is positioned vertically (north).


* Free software: MIT license
* Documentation: https://pdf-rotation.readthedocs.io.
